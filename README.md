## 付録

### A.1 マクロシステム
・テキスト置換マクロシステム
使用している言語とは別の言語を使って、使用している言語を生成したり、変更したりするマクロシステム。
例）C言語のプリプロセッサ

・構文マクロシステム
コードをデータとして扱う。
構文マクロが備わっている言語では外側のホスト言語だけでなく、その言語自身でソースコードを引き回したり、変更したり、生成したりできる。
例）Elixirのquote関数

quote()
評価を中止して、オブジェクトを返す。
unquote()
引数のquoteを迂回する。
macro()
マクロを定義する
